# Konstantin Orlov — Personal Portfolio

Персональный сайт-портфолио Константина Орлова, руководителя travel-функции.

**URL:** [konstantin-orlov.vercel.app](https://konstantin-orlov.vercel.app)

## Стек

- **Next.js 16** (App Router) — серверный рендеринг и маршрутизация
- **React 19** — UI-компоненты
- **Tailwind CSS 3.4** — стилизация без кастомных CSS-файлов
- **Vercel** — хостинг и автоматический деплой при пуше в `main`
- **FormSubmit.co** — отправка сообщений из контактной формы

## Структура проекта

```
app/
├── layout.js          — корневой layout, метаданные, шрифт Manrope
├── page.js            — главная страница (Hero, About, Services, Contact)
├── globals.css        — глобальные стили
├── contact-form.js    — клиентский компонент контактной формы
├── mobile-nav.js      — мобильная навигация (бургер-меню)
├── nav-links.js       — массив навигационных ссылок
├── robots.js          — генерация robots.txt
├── sitemap.js         — генерация sitemap.xml
└── llms.txt/
    └── route.js       — эндпоинт /llms.txt для AI-краулеров
```

## Что сделано

### Базовая версия (29 марта 2026)

- Создана one-page структура: Hero, «Обо мне», «Экспертиза», «Контакты»
- Минималистичный дизайн на Tailwind CSS
- Адаптивная вёрстка с мобильным бургер-меню
- Деплой на Vercel с автодеплоем из GitHub

### Доработки (9 апреля 2026)

- **Миграция на Next.js 16 + React 19** — обновлён стек с Next.js 14 / React 18
- **Рабочая контактная форма** — интеграция с FormSubmit.co, honeypot-защита от спама, обработка ошибок и статусов отправки
- **Модульная архитектура** — вынесены отдельные компоненты: `ContactForm`, `MobileNav`, `nav-links`
- **SEO-оптимизация:**
  - Полные метаданные (OpenGraph, Twitter Card, keywords, canonical)
  - JSON-LD structured data (Person + WebSite)
  - `robots.js` и `sitemap.js` для поисковых систем
  - `/llms.txt` — описание сайта для LLM-краулеров
- **Исправлена загрузка шрифтов** для корректной работы на Vercel

## План улучшений

### Высокий приоритет

- **Исправить конфликт шрифтов** — сейчас одновременно подключены Manrope (через `next/font`) и Inter (через CSS `@import`). Нужно оставить один шрифт и убрать лишний импорт
- **Добавить favicon** — в проекте отсутствует `favicon.ico` / `icon.png`, браузер показывает дефолтную иконку
- **Добавить OG-изображение** — без `opengraph-image` превью ссылки в Telegram, WhatsApp и соцсетях выглядит пустым
- **Подтвердить FormSubmit.co** — сервис требует одноразового подтверждения email, без этого форма не доставляет сообщения

### Средний приоритет

- **Улучшить мобильное меню** — добавить анимацию появления, закрытие по клику вне меню, блокировку скролла фона
- **Добавить scroll-анимации** — плавное появление секций при прокрутке (fade-up эффекты)
- **Динамический navbar** — изменение стиля навигации при скролле (тень, прозрачность)
- **Обновить project instructions** — привести в соответствие с текущим стеком (Next.js 16 вместо 14)

### Идеи на будущее

- Раздел «Кейсы» / «Проекты» с примерами реализованных задач
- Мультиязычность (EN / RU)
- Тёмная тема
